教程

第一次使用建议先看这里

基础使用流程

  1. 先在左侧选择月份。
  2. 进入「收入」「币种兑换率」「支出明细」分别填写数据。
  3. 主页会自动同步显示结余、差额、预算提醒与系统状态。
  4. 新月份会自动继承上个月的 plan，actual 会重置为 0。

JSON 导出 / 导入

  1. 点击顶部或左侧的「导出JSON」下载备份文件。
  2. 下载后请把 JSON 文件保存在安全位置，避免误删。
  3. 点击「导入JSON」时，系统会打开文件选择器；请选择你保存的备份文件或自动保存的 JSON 文件。
  4. 如果你使用自动保存，先打开你当初选择的文件夹，找到文件名为 budget-backup.json 的文件。
  5. 如果是手动导出的备份，一般会以 budget-backup-年月 命名，优先找最近的月份文件。
  6. 不知道文件在哪里时，可先检查「下载」文件夹、桌面、或你自己指定的自动保存文件夹。
  7. 导入后建议立刻再导出一次，保留最新备份。
提醒：导入会覆盖当前本机资料，执行前请先确认备份文件正确。自动保存文件通常固定在你选定的文件夹里，文件名为 budget-backup.json。

预算提醒与新系统

  1. 支出达到预算 80% 会显示黄色警告。
  2. 支出超过预算会显示红色超支提醒。
  3. 你现在还能使用分类图表、年度统计、资产负债、预算上限功能。
  4. 你现在也可以使用流水系统，记录收入、开支与资产转移，并自动计入对应项目。
  5. 导入自动保存文件时，先去你选定的固定文件夹里找 budget-backup.json。
  6. 找不到文件时，优先检查「下载」文件夹、桌面和你自己设置过的自动保存文件夹。
  7. 为了数据安全和方便使用，建议尽量启用自动保存功能。

欢迎使用个人月度预算管理系统 4.0

快速上手

  1. 选择月份后开始输入收入、汇率和支出。
  2. 主页会自动显示结余、差额、图表、预算提醒。
  3. 支出达到 80% 会黄色提醒，超过预算会红色提醒。
  4. 流水系统可把每一笔收入、开支或资产转移直接记到账目。

JSON 备份

  1. 点击「导出JSON」保存完整数据备份。
  2. 点击「导入JSON」恢复备份资料。
  3. 也可以选择一个文件夹启用自动保存，系统会自动写入 JSON 备份。
  4. 旧版本数据会依据版本号字段自动迁移到当前版本。
  5. 为了数据安全和使用方便，建议尽量启用自动保存功能，并定期保留一份手动导出的 JSON 备份。
  6. 如果更换浏览器、清除缓存或重装系统，自动保存文件会更稳妥。
  7. 支出明细的小项目名称可以直接修改，并会随 JSON 一起保存。
  8. 流水系统可记录每一笔收入、开支或资产转移的金额、备注、日期和时间。
  9. 导入自动保存文件时，请先打开你当初选择的固定文件夹，再选中 budget-backup.json。
  10. 如果找不到文件，可以先检查「下载」文件夹、桌面，或者你自己设置过的自动保存文件夹。
